# HarmonyTasks

鸿蒙任务项目。
所有人员拉取主分支后，创建新分支，新分支命名规则为姓名汉语拼音全拼，所有任务均只能在自己的分支上进行。勿合并到项目主分支。

4. 在commons层，创建CommonConstants公共常量管理类。（100%）
5. 实现product层引入commons层的方法、组件、静态资源文件。（100%）
